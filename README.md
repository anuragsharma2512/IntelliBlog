# IntelliBlog 📝

A modern AI-powered blogging platform built with React and FastAPI that helps users create, manage, and publish blogs effortlessly. IntelliBlog focuses on providing a clean writing experience, AI-assisted content generation, and efficient blog management.

## 🚀 Features
✍️ Create, edit, and delete blog posts
🤖 AI-powered content assistance
📚 View and manage all published blogs
🔍 Search and discover articles
🏷️ Categorize blog posts
🖼️ Upload and manage blog images
📱 Fully responsive user interface
⚡ Fast API-driven architecture
🎨 Modern and intuitive design
